### Task 1: Create Widget Background Drawable with Corner Radius
**type:** auto

**Action:**
Create `android/app/src/main/res/drawable/widget_background.xml` with 28dp corner radius and dark background.

```xml
<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android">
    <solid android:color="#121212" />
    <corners android:radius="28dp" />
</shape>
```

**Verification:**
- File exists at `android/app/src/main/res/drawable/widget_background.xml`
- Contains shape with solid color #121212
- Contains corners with 28dp radius

**Done:**
- Widget background drawable created with correct corner radius

### Task 2: Create START Button Background with Glow Effect
**type:** auto

**Action:**
Create `android/app/src/main/res/drawable/widget_start_button_background.xml` with orange background and shadow/glow effect.

```xml
<?xml version="1.0" encoding="utf-8"?>
<layer-list xmlns:android="http://schemas.android.com/apk/res/android">
    <!-- Glow/shadow layer -->
    <item android:top="0dp" android:left="2dp" android:right="2dp" android:bottom="4dp">
        <shape android:shape="rectangle">
            <solid android:color="#80FF6B00" />
            <corners android:radius="12dp" />
        </shape>
    </item>

    <!-- Main button layer -->
    <item>
        <shape android:shape="rectangle">
            <solid android:color="#FF6B00" />
            <corners android:radius="12dp" />
        </shape>
    </item>
</layer-list>
```

Also create a selector for pressed state: `android/app/src/main/res/drawable/widget_start_button_selector.xml`:

```xml
<?xml version="1.0" encoding="utf-8"?>
<selector xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:state_pressed="true">
        <shape android:shape="rectangle">
            <solid android:color="#E65C00" />
            <corners android:radius="12dp" />
        </shape>
    </item>
    <item>
        <bitmap android:src="@drawable/widget_start_button_background" />
    </item>
</selector>
```

**Verification:**
- Both drawable files exist
- Contains orange color #FF6B00
- Has shadow/glow effect (layer-list)

**Done:**
- START button background with glow effect created
- Pressed state selector created
