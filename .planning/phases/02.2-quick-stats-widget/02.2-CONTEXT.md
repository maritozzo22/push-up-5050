# Phase 2.2: Widget 1 - Quick Stats - Context

**Gathered:** 2026-01-21
**Status:** Ready for planning

## Phase Boundary

Implement Android home screen widget displaying today's and total push-up progress with a START button. Widget updates automatically when app state changes and provides tap-to-launch functionality.

---

## Implementation Decisions

### Layout & Sizing
- **Single size option**: 4x4 cells (standard size)
- **Default orientation**: Landscape (wider than tall)
- **Resize behavior**: Content scales proportionally when user resizes widget
- **Fixed structure**: Vertical stack — Header → 2-column stats section → START button

### Visual Styling
- **Match mockup exactly**: Colors, corner radius, typography as specified
- **START button effect**: Glowing orange (shadow/glow effect, not gradient)
- **Font family**: Montserrat throughout (matching app design system)
- **Logo icon**: Use app icon from `push_up_5050/Icone-App-Pushup/` (available in multiple densities: hdpi, ldpi, mdpi, xhdpi, xxhdpi, xxxhdpi)
- **Colors from mockup**:
  - Background: #121212 (dark)
  - Accent: #FF6B00 (orange)
  - Corner radius: 28px
  - Text: White (primary labels), Orange (OGGI/TOTALE headers)

### Content Display
- **Dynamic language**: Labels follow app's current language setting (Italian: OGGI/TOTALE, English: TODAY/TOTAL)
- **OGGI value**: Today's completed count only (e.g., "41" — not "41/50")
- **TOTALE value**: Current/total format with slash (e.g., "44 / 5050")
- **Layout**:
  - Left column: "OGGI:" (orange, bold) → "41" (white, large) → "push-ups" (white, small)
  - Right column: "TOTALE:" (orange, bold) → "44 / 5050" (white, large) → "push-ups" (white, small)
  - Bottom: Orange rectangular button with "START" text (white, bold)

### Tap Behavior
- **START button**: Opens SeriesSelectionScreen
- **Widget body**: No tap action — only START button is interactive
- **Navigation mechanism**: You decide (deep link vs pending intent)

### Claude's Discretion
- Exact navigation mechanism (deep link vs Android pending intent)
- Icon size selection from available densities
- Proportional scaling implementation details
- Glow effect implementation on START button
